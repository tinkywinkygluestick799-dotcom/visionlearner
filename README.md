# VisionLearner

CNN vs Random Forest on MNIST and CIFAR-10.

Result: CNN achieved 99.3% accuracy on MNIST.

Run:
git clone https://github.com/tinkywinkygluestick799-dotcom/visionlearner
cd visionlearner
pip install -r requirements.txt
python train_cnn.py

Dependencies: Python 3.8+, TensorFlow, scikit-learn, Matplotlib
